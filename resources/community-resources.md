Inclusion does not imply endorsement.

## Using community sources

Community resources can be useful for identifying:

- common scent impressions;
- performance opinions;
- frequently discussed comparisons;
- popularity trends;
- packaging changes.

Community opinions should not be presented as verified facts. Official product information should be preferred for release details, concentration, bottle sizes, and note descriptions.
