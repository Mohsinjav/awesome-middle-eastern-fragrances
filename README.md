# Awesome Middle Eastern Fragrances

A community-maintained directory of Middle Eastern fragrance brands, notable perfumes, trusted retailers, buying guides, fragrance terminology, and useful resources.

The objective of this repository is to make reliable information about Middle Eastern perfumery easier to find and understand.

> This repository is independently maintained and is not affiliated with every brand or retailer listed. Inclusion does not constitute an endorsement.

## Inclusion criteria

A brand, retailer, or resource should generally meet the following requirements:

1. It must be relevant to Middle Eastern or Arabian perfumery.
2. An official or reputable source should support information.
3. Descriptions must remain factual and non-promotional.
4. Affiliate and tracking links are not allowed.
5. Retailers should have a functioning website and clearly published business policies.
6. Community fragrance comparisons must be presented as opinions, not facts.
